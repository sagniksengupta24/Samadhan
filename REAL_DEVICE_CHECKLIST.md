# Real-Device Readiness Checklist

Use this checklist on Expo Go or an Android device before an MVP demo.

- App opens without a red error screen.
- Sign in works with an existing confirmed Supabase test user.
- Blank Home state loads clearly when there are 0 properties.
- Saved, List, and Mine redirect signed-out users to auth.
- List Property / create form opens after sign-in.
- Image picker opens from the property media slots.
- Sign out works from My Listings.
